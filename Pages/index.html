Welcome

Tired of using Scratch and want to learn something new? Then this is for you! This tutorial website aims to help you get started in learning new programming languages.

So let's get started!


Instead of using Scratch to create game projects, you will now be using Visual Studio Code.

Visual Studio Code is a code editor that helps you write and test your code.




To begin, you must download and install Visual Studio Code.




Let's take a look below and see the differences between Scratch and Visual Studio Code.

What Scratch look like

In Scratch, you start your project like this. You drag and drop the Scratch blocks you need into the Script editing area.

What Visual Studio Code look like

In Visual Studio Code, you start your project like this. Code are now in text rather than in blocks. You write your code inside the editor.






By default, the background theme of Visual Studio Code is Dark.

To change it, Go to File > Preferences > Color Theme
